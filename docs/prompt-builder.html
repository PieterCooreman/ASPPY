ASPPY Prompt Builder

Generate a complete project prompt for AI coding agents

What is this?

This tool builds a structured JSON prompt that contains your project description together with every technical rule from the ASPPY documentation. You paste the JSON output into a coding agent like Claude Code, Cursor, or GitHub Copilot as your first message.

Why it works well: AI coding agents make most of their mistakes early, when they lack context about the framework. A single structured prompt front-loads the hard rules, path conventions, MVC structure, known pitfalls, and your project requirements into one message. The agent reads everything before writing a single line of code.
